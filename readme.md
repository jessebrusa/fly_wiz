# Marketing Flyer Creator "Fly Wiz"

## Overview
Marketing Flyer Creator is an intuitive application designed for creating professional marketing flyers. The app features a user-friendly graphical interface and six distinct layout options, making it easy for users to design compelling marketing materials.

---

## Features

- **Graphical User Interface (GUI):** Simple and intuitive design for ease of use.
- **Six Unique Layouts:** Choose from six different pre-designed templates.
- **Save and Load Projects:** Save your progress and reload your projects anytime.
- **Export Options:** Export your flyers in various formats (e.g., PDF, JPEG, PNG).
